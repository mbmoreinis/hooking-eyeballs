What is Motion Design?

Motion design is a great tool that many talented people use. It is the act of creating motion graphics and designs, such as the photo shown above. From creating animated videos, and images; learning about motion design really could an awesome experience! This blog will be talking about the different file formats that you are able to export your work to.

Different File Formats

There are many different ways to export your animated work, but the two most common are:

  • GIF (Graphics Interchange Format) [Officially pronounced: JIF]
  • WebP (Web Picture Format) [Pronounced as is]

.gif [V] .webp

Both of these file formats are capable of exporting animated images, but the most popular and widely supported animated file format is GIF. On the other hand, WEBP files have smaller file sizes, support more colors (2^24=16,700,000), has better image transparency, and supports both lossy and lossless compression.
